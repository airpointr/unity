Detailed Description

Control a pointer on the screen. Provides projection modes for GearVR and Cardboard.

For setup with GearVR please follow these steps:

  • In Player Settings you have to check 'Virtual Reality Supported' under "Other Settings". Also set the 'Minimum API required' to 4.4 (needed for GearVR).
  • You need to set the Projection Mode of the ScreenPointer script to GearVRSize. You also have to change the Canvas Render Mode to 'Screen Space Camera'.
  • 'Render Camera' must be set to 'CenterEyeAnchor' Camera from the "OVRCameraRig" Prefab found in OculusUtilities package for Unity.

For setup with Cardboard please follow these steps:

  • You can use either "CardboardHead" or "CardboardMain" Prefab with the "CardboardScreenPointr" Prefab.
  • You have to set the 'Render Camera' for both 'CanvasLeft' and 'CanvasRight' (inside the 'CardboardScreenPointr').
  • You can set them by Drag&Drop or select either 'Main Camera Left/Right' or 'Camera Left/Right' according to the Cardboard-Prefab you use.
