AirPointr - Unity Asset Documentation

Unity Asset

AirPointr is a gesture control plugin for using hands to control your mobile device. Add touchless gestures to control your game or application using just the built-in camera. Utilizes advanced scene change detection to provide a stable and precise gesture interface.

The manual as well as the latest version of this document is available from https://www.airpointr.com/unity.

You can also download the CHM of this documentation. The file might be blocked because it has been downloaded from the internet. To unblock, simply right-click on the CHM file in explorer and select the 'Properties' menu item: Then, click the 'Unblock' button near the bottom of the properties page: You can then open the file and should see the first page in the file.
